# **App Name**: Opita Club Manager

## Core Features:

- Role-Based Access: Implement different dashboards for admins, managers, coaches, and athletes, each with role-specific functionalities.
- Coach Management Panel: Coaches can manage athlete information, including contact details, emergency contacts, and medical information.
- Manager Overview Dashboard: Provides a high-level overview of club operations, including athlete enrollment, payment status, and task assignments.
- Task Assignment: Allows managers to assign tasks to coaches and staff, track progress, and set deadlines.
- Athlete Payment Registration: Athletes can register their monthly payments through their dashboard.
- Payment Management: The app stores the payment data using an external data base for each member.
- Informative Landing Page: Landing page showcasing club information, registration details, upcoming events, and news.

## Style Guidelines:

- Primary color: Forest Green (#4CAF50) to reflect growth, health, and the club's focus on sports.
- Background color: Off-White (#F5F5F5), providing a clean, professional, and modern look with a very subtle green tint.
- Accent color: Teal (#008080) for interactive elements and important notifications, providing contrast and visibility.
- Headline font: 'Poppins' sans-serif, for headlines and short amounts of body text; geometric with a contemporary feel.
- Body font: 'PT Sans' sans-serif, for body text, providing a modern look with a little warmth or personality; this pairs well with Poppins, should the text require more explanation than labels
- Use consistent, modern icons to represent different features and sections of the app.
- Maintain a clean and organized layout with clear visual hierarchy to ensure easy navigation.
